1.HJB equation de related problem :

Dans cette partie dans l'article (chapitre 3) on considre le problme d'arret optimal afin de pricer une option amricain.

Ce problme d'arret optimal est convertie en un problme de controle avec l'equation HJB (3.2) :

$\partial_t V^\lambda + \mathcal{L}_S V^\lambda + \lambda \exp\left(-\frac{V^\lambda - g}{\lambda}\right) = 0, \quad V^\lambda(S,T) = g(S).$

Thorme 3.7 : La solution de cette EDP 3.2 converge vers le prix standard d'une option amricain quand le terme de rgularisation $\lambda$ tend vers zero

$u^\lambda$ --> $u^*$ qd $\lambda->0$

Illustration du Thorme 3.7 :

On cherche la solution de l'EDP 3.2 $u^\lambda$ en utilisant la discrtisation de l'EDP pour avoir shema d'euler implicite combine avec la mthode de newtone. (le solver est impliment dans la fonction pde_solver_regularization dans le fichier solvers.py)

En ce qui concerne le prix du l'option amrcain standard $u^*$ on peut le calculer en utilisant la mthode de pnalit Penalty Method rfr par l'article dans [9]

Bref explication du Penalty method :

On considre la valeur de l'option $V(S,t)$ satisfaisant l'inquation variationnelle : \begin{equation} \max \left( \partial_t V + \mathcal{L}_S V + f, V - g \right) = 0, \end{equation}

La mthode de pnalisation approxime cette inquation par une EDP modifie : \begin{equation} \partial_t V^\lambda + \mathcal{L}_S V^\lambda - r V^\lambda - \lambda \max(0, g(S) - V^\lambda) = 0, \end{equation} Ici, $\lambda > 0$ est un paramtre de pnalisation qui impose progressivement la contrainte $V \geq g$ lorsque $\lambda \to 0$. La solution $V^\lambda$ converge alors vers la solution de l'option amricaine.

On va utiliser le meme shma numrique pour rsoudre les deux EDP

2. Solving the HJB equation using the policy iteration :

En apprentissage par renforcement, une mthode pour apprendre la stratgie optimale est l'itration de politique, dans laquelle la stratgie optimale est approxime par une mise jour itrative. Plus prcisment, tant donne une stratgie de retour $\pi^n(x,t)$, la fonction de valeur correspondante $u^n(x,t)$ satisfait

\begin{equation} \partial_t u^n + \mathcal{L}_x u^n + H(x, \pi^n(x,t), u^n) = 0, \quad u^n(x,T) = h(x), \end{equation}

o l'Hamiltonien $H$ est dfini comme

\begin{equation} H(x, \pi, u) = (h(x) - u)\pi + \lambda R(\pi). \end{equation}

Ayant une fonction de valeur $u^n$, on peut construire une stratgie de retour $\pi^{n+1}$ comme $ \begin{equation} \pi^{n+1}(x,t) = \arg\max_{\pi \in (0,\infty)} H(x, \pi, u^n(x,t)) = \exp \left( -\frac{u^n(x,t) - h(x)}{\lambda} \right). \end{equation} $

Theorme 3.3:

Soit une estimation initiale $u^0 \in C_b(\mathbb{R})$ pour la fonction de valeur optimale. La suite $\{u^n, \pi^n\}_{n=1,2,\dots}$ est dfinie itrativement selon \eqref{eq:policy_update} et \eqref{eq:policy_iteration}. Alors, nous avons

$u^\lambda \geq u^{n+1} \geq u^n, \quad \text{pour } n = 1, 2, \dots$

On s'attend ce que les fonctions de valeur $\{u^n\}$ convergent vers la fonction de valeur optimale $u^\lambda$. C'est effectivement le cas. De plus, nous pouvons donner une estimation du taux de convergence des fonctions de valeur.

Thorme 3.4 :

Soit $M := \|(h - u^1)^+\|_{\infty}$. Il existe une constante $C$ indpendante de $\lambda$ telle que : $\| u^{n+1} - u^\lambda \|_{\infty} \leq C^n \frac{T^n}{n!} \exp \left( n \frac{M}{\lambda} + rT \right) (M + K + \lambda T).$
